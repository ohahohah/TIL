## Type Conversion
- `print(str(2)+ str(5))` >> 25
- `print(int(3.8))` >> 3
- `print(float(3))` >> 3.0
- `print(int("2")+ int("5"))` >> 7


## 단순 문법
- `print(2 ** 3)` >> 8 / `print(2 ** 3.0)` >> 8.0
- `print(4 / 2)`  >> 2.0 : 나눗셈 연산 무조건 결과값 floating point
- `print("Hello" * 3)` >> HelloHelloHello
- `print("%d일 %s요일. %.2f"" % (day + 1, "월", (1.0 / 3)))` >> 13일 월요일. 0.33
